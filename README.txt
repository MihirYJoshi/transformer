This code has encoder classes that are used to classify which president between Obama, Bush Sr, and Bush Jr. made a statement
It also has the Decoder class that generates more text in the styles of Obama, Bush sr, and Bush Jr

To run the Encoder, run: python3 main.py part1
To run the Decoder, run: python3 main.py part2

If you use different parameters when running the function, it will return an error.

By default, both run the sanity check on a sentence that is provided in the code.
